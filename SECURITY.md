# Security Policy

## Supported Versions

Security fixes are handled for the latest public release.

## Reporting A Vulnerability

Please report security issues privately by email to:

```text
dev@dariocorsetti.com
```

Include:

- the affected version or commit,
- the platform and GnuCash version,
- steps to reproduce,
- the impact you believe the issue has,
- whether a GnuCash book, CSV file, or rule file is needed to reproduce it.

Do not include private financial data. Use a reduced sample book or anonymized CSV whenever possible.
